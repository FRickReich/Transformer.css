Transformer.css

A Lightweight CSS wrapper for easy css-only animations.

Transformer.css is a css wrapper to create css-only animations, without the use of javascript. The animations are defined by classes inside your html element.

Using Transformer.css is easy, implement transformer.css into your html file and add the css wrapper classes to your div, p, li or whatever element you want to animate.

For example, the code of the cube below looks like this:
<div class="cube rotate360cw time2s alternate infinite step-end"></div>

cube is just the elements styling class, you can ignore it.
rotate360cw makes the cube rotate 360° clockwise around its own axis.
time2s defines a length of 2 seconds for the animation.
alternate sets the animation to alternate moving that rotates and returns again.
infinite sets the amount of rotations to infinite.
step-end makes the animation use a step-end curve, resulting in a rapid movement.

It is mandatory that you always use all classes to define the animation, otherwise it will not work.

Animation Degrees

The degree of an animation is defined by the .rotate class in steps of 10.

rotate90cw or rotate120cw for example rotate the html element by 90° or 120° clockwise, while rotate30ccw for example rotates the element 30° counter-clockwise.

Animation Length

The length of the animation is defined by the .time class in steps of 1 second, with a maximum of 10 seconds.

time1s will make the animation run for 1 second, while time5s will make the animation run for 5 seconds.

Animation Direction

The direction of the animation is defined by the following classes:

normal the animation goes into its normal direction.
reverse the animation goes backwards.
alternate The animation goes an alternative direction

Animation Steps

The animation steps are defined as either infinite or an amount of 1 - 10.

infinite makes the animation repeat itself forever.
times1 tells the animation to run X amount of times.

Animation Transistion

The animation transistions change the appeal of the animation. For example if it goes linear all the way through, or starts slow and ends fast. linear makes the animation go at a steady speed
ease-in makes the animation start slow and end fast
ease-in-out starts slow, gains speed and slow down again
ease-out makes the animation start fast and end slow
